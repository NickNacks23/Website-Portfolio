My Interests

Web/Mobile Development

I'm passionate about developing and creating responsive and easily accessible web applications that provide the smoothest experience for all of the users on any device. I have worked with React, HTML, CSS, and JavaScript to build projects that best focus on usability and performance. I am also interested in learning more about mobile design and how to best optimize websites for real-world users.

UI/UX Design

I really enjoy the process of having the freedom to design interfaces that are not only intuitive and visually pleasing, but also suitable for all the users that come across it. I have used other tools like Canva or Figma to help me focus on the layout structures, color balance, and obtaining a clear user flow. I truly believe that a good design should have the ability to feel effortless to use while looking great to a user’s eyes.

Front-end Development

The reason I am so drawn to front-end development is because it holds a connection between the use of my creativity and logic. As said before, I use React, JavaScript, and CSS frameworks like Tailwind to help me build dynamic and interactive interfaces. My pride comes from the ability to write clean, maintainable code that eventually will turn my design concepts into functional, polished, creatively fun products.
